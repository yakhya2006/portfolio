Mon stage

Mission principale : refonte du site web de l'entreprise.

Stage BTS SIO · 2026

Refonte du site Avalon

Avalon — Terres de Jeux et d'Utopies · Le Landreau (44)

Refonte complète du site web de l'association Avalon, terrain d'airsoft et d'activités outdoor, en passant de Wix à un site WordPress sur-mesure. J'ai développé un thème PHP custom avec Advanced Custom Fields (gratuit) pour que le responsable puisse modifier le contenu sans toucher au code. Le site est hébergé sur un VPS OVH via Docker.

  • Thème WordPress entièrement codé à la main (PHP)
  • Champs éditables via ACF (logo, hero, coordonnées, réseaux)
  • 4 types de contenu personnalisés : Activités, Tarifs, Sessions, Terrains
  • Déploiement Docker sur VPS OVH
